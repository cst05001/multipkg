#安装说明

你可以直接下载我已经打包好的RPM。如果有洁癖可以按一下方法从源码安装。

##安装打包软件 multipkg

###从官网安装

[https://github.com/ytoolshed/multipkg][1]

###直接下载我打包好的

[https://github.com/cst05001/multipkg][2]

##用multipkg打包本软件

    git-multipkg -b https://github.com/cst05001 f8n-multipkg/centos7

打完，收工。
